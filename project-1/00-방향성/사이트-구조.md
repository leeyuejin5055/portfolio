# 사이트 구조

## 페이지
- `/` Home — 히어로 + Selected work
- `/work` — 프로젝트 목록
- `/work/[slug]` — 케이스 스터디
- `/about` — 소개 + 연락처

## 콘텐츠 위치
- 프로젝트 MD: `web/src/content/projects/`
- 커버 이미지: `web/public/images/projects/`
- 사이트 메타: `web/src/data/site.ts`
