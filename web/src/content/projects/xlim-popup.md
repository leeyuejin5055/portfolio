## Overview

서울 기반 패션 브랜드 XLIM의 브랜드 무드를 분석해 제작한 팝업 포스터입니다.

암석, 바람, 지형, 원단의 절개와 패턴에서 영감을 받아 자연의 형태와 의복 구조를 콜라주 방식으로 결합했습니다.

패션 브랜드의 정체성이 약해지지 않도록 실제 의상의 실루엣, 소재, 절개선이 그래픽 안에서 충분히 드러나도록 구성했습니다.

## Key Points

- 패션 브랜드 무드 분석
- 자연 지형과 의복 구조의 연결
- 원단, 절개, 실루엣을 활용한 그래픽 콜라주
- 팝업 스토어 정보 설계
- 실험적인 에디토리얼 레이아웃
